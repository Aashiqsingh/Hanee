About C Lnaguage

C is a high-level, general-purpose programming language that's used to develop computer software, applications, and systems. It was originally developed in 1972 by Dennis Ritchie at Bell Telephone Laboratories for the UNIX operating system, and is now one of the most widely used programming languages.
